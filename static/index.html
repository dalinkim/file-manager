File Organizer App

This app is to help you organize your files in a directory.
Based on the keyword(s) you provide, new subfolder(s) will be created which contain(s) the copy of any matching files.
A file is considered 'matched' if its file name and/or its content contains the keyword provided.
The following file types are reviewed for matching: docx, jpg, pdf, png, pptx, and txt. (Files with no content are ignored.)
